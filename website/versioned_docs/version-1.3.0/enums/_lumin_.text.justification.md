Describes how text in a text region (e.g., paragraph) is justified.

## Enumeration members

###  kCentered

• **kCentered**:

All lines are centered with respect to the text-region boundary.

___

###  kEnd

• **kEnd**:

All lines end at the last position (i.e., right-justified for English).

___

###  kStart

• **kStart**:
