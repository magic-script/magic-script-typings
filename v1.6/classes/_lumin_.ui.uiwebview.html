Class UiWebView

UiWebView - This node renders web content (HTML, css, javascript). and provides a simiplified browsing feature set.

To embed web content: Create and add this node into the application node tree. To receive notifications from UiWebView: Set a web view client using 'setWebViewClient'. UiWebView will notify errors or statechanges.
